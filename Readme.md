# Sovereign Financial Analyst

A **privacy-first AI financial research assistant** for analyzing **10-K filings and stock performance** using a local LLM and agentic Retrieval-Augmented Generation (RAG).

Sovereign enables financial analysts to quickly extract insights from SEC filings by combining:

- Vector search over company filings
- Tool-enabled LLM reasoning
- Live stock market data

All processing runs **locally** — no external APIs are required for LLM inference.
